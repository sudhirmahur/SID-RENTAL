# SID-RENTAL

SID-RENTAL is a web-based platform designed to facilitate vehicle rentals, offering a seamless experience for both renters and owners. The platform allows users to browse available vehicles, book rentals, and manage their rental history. Owners can list their vehicles and track bookings, while the website ensures smooth transactions and communication between both parties.

## Features

- **Vehicle Listing**: Owners can list their vehicles for rent.
- **Booking System**: Renters can easily browse vehicles and book them.
- **User Authentication**: Secure login for both renters and owners.
- **Booking Management**: View and manage current and past bookings.
- **Payment Integration**: Safe and secure online payment for rental bookings.
- **Review System**: Renters can rate and review vehicles they have rented.
- **Responsive Design**: Optimized for both desktop and mobile use.

## Technologies

- **Backend**: Node.js, Express.js
- **Frontend**: React.js, HTML, CSS, JavaScript
- **Database**: MongoDB (Mongoose for schema modeling)
- **Authentication**: JWT (JSON Web Tokens)
- **Payment**: Stripe API integration
- **Deployment**: Docker, Nginx, AWS EC2
